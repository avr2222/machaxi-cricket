How It Works

This tool reads all 12 match scorecard PDFs from the MatchScoreCards/ folder, extracts batting and bowling data from each innings, and produces three downloadable CSVs.

It uses PDF.js 3.11.174 to parse text with positional (x,y) data directly in your browser — no server required (open via http://, not file://).
